# Customer Churn Prediction

## Business Problem

Customer churn leads to revenue loss.  
This project predicts which customers are likely to churn so proactive actions can be taken.

## Project Overview

- End-to-end machine learning pipeline
- Churn prediction using classification models
- Threshold optimization for better recall
- Deployed with Streamlit

## Model Performance

- ROC-AUC: 0.84
- Accuracy: 0.79
- Recall optimized for churn detection

## Model Comparison

| Model                | ROC-AUC |
|---------------------|--------|
| Logistic Regression | 0.84   |
| Random Forest       | 0.81   |

## Deployment

Streamlit app allows real-time prediction:

- Input customer data
- Get churn probability and risk level

## Technologies

- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit
